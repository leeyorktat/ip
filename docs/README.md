# Duke User Guide

## Features 

### Adding a todo task
Adds a todo task to your list of tasks.
Format: `todo DESCRIPTION`

Example of usage: 

`todo return book`

Expected outcome:

`Added this task to your list:`
`[T][✘] return book`
`You now have 4 task(s) in the list.`
